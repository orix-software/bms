## unsigned int bms_seek(bms *bms, unsigned int offset, unsigned char whence);


***Description***

seek to offset

***Input***

* bms (bms *) bms struct
* offset (unsigned int) data
* whence (unsigned char) (possible value : BMS_SEEK_CUR, BMS_SEEK_SET, not managed : BMS_SEEK_END,)

***Returns***

* (unsigned int) result
